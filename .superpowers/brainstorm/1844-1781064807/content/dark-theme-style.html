1. 两数之和
简单 数组 哈希

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出和为目标值 target 的那两个整数，并返回它们的数组下标。

输入：nums = [2,7,11,15], target = 9

输出：[0,1]

C++ Java Python
▶ 运行 提交
def twoSum(nums, target):
  seen = {}
  for i, n in enumerate(nums):
    diff = target - n
    if diff in seen:
      return [seen[diff], i]
    seen[n] = i
