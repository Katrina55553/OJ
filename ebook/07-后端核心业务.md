## 7.1 用户系统

### 注册流程

```java
// UserServiceImpl.java
@Override
public long userRegister(UserRegisterRequest request) {
    String account = request.getUserAccount();
    String password = request.getUserPassword();
    String confirmPassword = request.getCheckPassword();

    // 1. 参数校验
    if (StringUtils.isAnyBlank(account, password, confirmPassword)) {
        throw new BusinessException(ErrorCode.PARAMS_ERROR, "参数为空");
    }
    if (account.length() < 4) {
        throw new BusinessException(ErrorCode.PARAMS_ERROR, "账号长度不能小于 4");
    }
    if (password.length() < 8) {
        throw new BusinessException(ErrorCode.PARAMS_ERROR, "密码长度不能小于 8");
    }
    if (!password.equals(confirmPassword)) {
        throw new BusinessException(ErrorCode.PARAMS_ERROR, "两次密码不一致");
    }

    // 2. 账号查重
    QueryWrapper<User> queryWrapper = new QueryWrapper<>();
    queryWrapper.eq("userAccount", account);
    long count = userMapper.selectCount(queryWrapper);
    if (count > 0) {
        throw new BusinessException(ErrorCode.PARAMS_ERROR, "账号已存在");
    }

    // 3. 密码加密（MD5）
    String encryptPassword = DigestUtils.md5DigestAsHex(password.getBytes());

    // 4. 入库
    User user = new User();
    user.setUserAccount(account);
    user.setUserPassword(encryptPassword);
    user.setUserRole(UserRoleEnum.USER.getValue());
    boolean save = this.save(user);
    if (!save) {
        throw new BusinessException(ErrorCode.SYSTEM_ERROR, "注册失败");
    }
    return user.getId();
}
```

### 登录流程

```java
@Override
public LoginUserVO userLogin(UserLoginRequest request, HttpServletRequest httpRequest) {
    String account = request.getUserAccount();
    String password = request.getUserPassword();

    // 1. 查询用户
    QueryWrapper<User> queryWrapper = new QueryWrapper<>();
    queryWrapper.eq("userAccount", account);
    User user = userMapper.selectOne(queryWrapper);
    if (user == null) {
        throw new BusinessException(ErrorCode.PARAMS_ERROR, "账号不存在");
    }

    // 2. 校验密码
    String encryptPassword = DigestUtils.md5DigestAsHex(password.getBytes());
    if (!encryptPassword.equals(user.getUserPassword())) {
        throw new BusinessException(ErrorCode.PARAMS_ERROR, "密码错误");
    }

    // 3. 检查是否被封禁
    if (UserRoleEnum.BAN.getValue().equals(user.getUserRole())) {
        throw new BusinessException(ErrorCode.NO_AUTH_ERROR, "账号已被封禁");
    }

    // 4. 生成 JWT
    String token = JwtUtils.createToken(user.getId());

    // 5. 返回脱敏用户信息
    return getSafetyUser(user, token);
}
```

### 用户信息脱敏

```java
private LoginUserVO getSafetyUser(User user, String token) {
    LoginUserVO loginUserVO = new LoginUserVO();
    loginUserVO.setId(user.getId());
    loginUserVO.setUserAccount(user.getUserAccount());
    loginUserVO.setUserName(user.getUserName());
    loginUserVO.setUserAvatar(user.getUserAvatar());
    loginUserVO.setUserProfile(user.getUserProfile());
    loginUserVO.setUserRole(user.getUserRole());
    loginUserVO.setCreateTime(user.getCreateTime());
    loginUserVO.setToken(token);
    // 注意：不返回 userPassword 等敏感字段
    return loginUserVO;
}
```

## 7.2 题目管理

### 数据模型

@TableName("question")
public class Question {
    @TableId(type = IdType.AUTO)

    private String title;

### 分页查询

```java
@Override
public Page<Question> listQuestionByPage(QuestionQueryRequest request) {
    // 1. 构建分页对象
    Page<Question> page = new Page<>(request.getCurrent(), request.getPageSize());

    // 2. 构建查询条件
    QueryWrapper<Question> queryWrapper = new QueryWrapper<>();
    queryWrapper.like(StringUtils.isNotBlank(request.getTitle()),
                      "title", request.getTitle());
    queryWrapper.eq(StringUtils.isNotBlank(request.getDifficulty()),
                    "difficulty", request.getDifficulty());
    queryWrapper.like(StringUtils.isNotBlank(request.getTags()),
                      "tags", request.getTags());
    queryWrapper.orderByDesc("createTime");

    // 3. 执行分页查询
    return questionMapper.selectPage(page, queryWrapper);
}
```

### VO 脱敏

题目详情返回给前端时，**不包含 answer 字段**（防止作弊）：

```java
@Override
public QuestionVO getQuestionVO(Question question) {
    QuestionVO vo = new QuestionVO();
    BeanUtils.copyProperties(question, vo);
    vo.setAnswer(null);  // 脱敏：不返回答案
    // 解析 JSON 字段
    vo.setJudgeConfig(parseJudgeConfig(question.getJudgeConfig()));
    vo.setTags(parseJsonArray(question.getTags(), String.class));
    return vo;
}
```

## 7.3 代码即时执行

```java
@RestController
@RequestMapping("/code")
public class CodeExecuteController {

    @Autowired
    private DockerCodeSandbox dockerCodeSandbox;

    @PostMapping("/execute")
    public BaseResponse<ExecuteCodeResponse> executeCode(
            @RequestBody ExecuteCodeRequest request) {
        // 直接调用沙箱执行，同步返回结果
        // 区别于判题：不经过 MQ，不更新数据库
        ExecuteCodeResponse response = dockerCodeSandbox.executeCode(request);
        return ResultUtils.success(response);
    }
}
```
