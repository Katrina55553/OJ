## 5.1 暗色主题系统

### 设计思路

本项目采用 **GitHub Dark + VS Code** 风格的暗色主题，技术方案：

1. **CSS 变量覆盖** — 重写 Arco Design 的 CSS 变量
2. **`arco-theme="dark"`** — 触发 Arco 内置暗色模式
3. **Pinia 状态管理** — 全局主题状态
4. **localStorage 持久化** — 记住用户选择
5. **系统偏好检测** — 自动跟随系统主题

### Pinia Theme Store

```typescript
// src/store/theme.ts
import { defineStore } from "pinia";
import { ref } from "vue";

export const useThemeStore = defineStore("theme", () => {
  // 初始化逻辑：
  // 1. 优先读取 localStorage
  // 2. 其次检测系统偏好
  // 3. 默认亮色
  const isDark = ref<boolean>(
    localStorage.getItem("theme") === "dark" ||
      (!localStorage.getItem("theme") &&
        window.matchMedia("(prefers-color-scheme: dark)").matches)
  );

  const toggleDark = () => {
    isDark.value = !isDark.value;
    applyTheme();
  };

  const setDark = (value: boolean) => {
    isDark.value = value;
    applyTheme();
  };

  const applyTheme = () => {
    // 1. 设置 body 的 arco-theme 属性
    if (isDark.value) {
      document.body.setAttribute("arco-theme", "dark");
    } else {
      document.body.removeAttribute("arco-theme");
    }
    // 2. 持久化到 localStorage
    localStorage.setItem("theme", isDark.value ? "dark" : "light");
  };

  // 初始化时立即应用
  applyTheme();

  // 监听系统主题变化（仅在用户未手动选择时生效）
  window.matchMedia("(prefers-color-scheme: dark)").addEventListener("change", (e) => {
    if (!localStorage.getItem("theme")) {
      isDark.value = e.matches;
      applyTheme();
    }
  });

  return { isDark, toggleDark, setDark };
});
```

### CSS 变量覆盖

```css
/* src/assets/dark-theme.css */

/* 全局基础 */
html, body, #app {
  background: #0d1117;
  color: #c9d1d9;
}

/* Arco Design CSS 变量覆盖 */
:root {
  /* 背景色系 */
  --color-bg-1: #0d1117;    /* 最深背景 */
  --color-bg-2: #161b22;    /* 卡片背景 */
  --color-bg-3: #21262d;    /* 输入框背景 */
  --color-bg-4: #30363d;    /* 悬浮背景 */
  --color-bg-5: #484f58;    /* 禁用背景 */

  /* 文字色系 */
  --color-text-1: #f0f6fc;   /* 主要文字（最亮） */
  --color-text-2: #c9d1d9;   /* 次要文字 */
  --color-text-3: #8b949e;   /* 辅助文字 */
  --color-text-4: #6e7681;   /* 最暗文字 */

  /* 边框 */
  --color-border: #30363d;
  --color-border-2: #21262d;

  /* 主色调（蓝色） */
  --color-primary-6: #58a6ff;
  --color-primary-5: #79c0ff;
  --color-primary-7: #388bfd;

  /* 功能色 */
  --color-success-6: #3fb950;  /* 绿色（通过） */
  --color-warning-6: #d29922;  /* 橙色（警告） */

}

.arco-modal { background: #161b22 !important; }
.arco-dropdown-menu { background: #161b22 !important; }
.arco-pagination { background: #161b22 !important; }
```

### 颜色体系

| 用途 | 颜色 | 示例 |
|------|------|------|
| 最深背景 | `#0d1117` | 页面背景 |
| 卡片背景 | `#161b22` | 卡片、菜单 |
| 输入框背景 | `#21262d` | 输入框、选择器 |
| 主要文字 | `#f0f6fc` | 标题 |
| 次要文字 | `#c9d1d9` | 正文 |
| 辅助文字 | `#8b949e` | 描述 |
| 主色调 | `#58a6ff` | 链接、高亮 |
| 成功 | `#3fb950` | 通过状态 |
| 警告 | `#d29922` | 中等难度 |
| 危险 | `#f85149` | 失败状态 |

### ThemeSwitcher 组件

```vue
<!-- src/components/ThemeSwitcher.vue -->
<template>
  <a-button class="theme-switcher" shape="circle" @click="themeStore.toggleDark()">
    <template #icon>
      <icon-sun v-if="themeStore.isDark" />
      <icon-moon v-else />
    </template>
  </a-button>
</template>

<script setup lang="ts">
import { useThemeStore } from "@/store/theme";
const themeStore = useThemeStore();
</script>
```

一行代码集成到任意页面：`<ThemeSwitcher />`
