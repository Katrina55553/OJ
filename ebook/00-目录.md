# OJ 在线判题系统 — 全栈项目实战教程

> 从零到一，深入理解一个生产级 Online Judge 系统的前后端全栈实现。

## 项目简介

本教程以一个完整的 **在线判题系统（Online Judge）** 项目为蓝本，涵盖 Vue 3 前端、Spring Boot 后端、Docker 容器化部署的全链路技术栈。项目支持多语言代码提交与自动判题、暗色主题等特性，适合全栈开发者学习参考。

## 技术栈速览

| 层 | 技术 |
|---|------|
| 前端 | Vue 3、TypeScript、Vite、Arco Design Vue、CodeMirror 6、ByteMD |
| 后端 | Spring Boot 3.2、MyBatis-Plus、JWT |
| 数据库 | MySQL 8.0、Redis 7 |
| 消息队列 | RabbitMQ |
| 代码沙箱 | Docker 容器隔离（每语言独立 Dockerfile） |
| 部署 | Docker Compose、Nginx、GitHub Actions |

## 如何使用本教程

1. **按顺序阅读**：从第 1 章到第 17 章，完整走一遍项目全貌
2. **按需跳转**：根据目录直接跳到感兴趣的章节
3. **配合源码**：每章都有大量代码引用，建议边读边对照实际代码
4. **动手实践**：第 2 章的环境搭建是后续所有章节的基础，务必先完成
